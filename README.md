# IFSHEN2002 个人主页

一个现代化的响应式个人主页，展示个人技能、项目和联系方式。

## 功能特性

- 🎨 现代化设计，响应式布局
- 📱 移动端友好，适配各种设备
- ⚡ 流畅的动画效果和交互
- 🎯 单页面应用，快速加载
- 📧 联系表单功能
- 🎭 打字机效果和滚动动画
- 🌈 渐变背景和现代化UI

## 部署说明

### 本地开发
1. 下载所有文件到本地目录
2. 用浏览器打开 `index.html`
3. 或者使用本地服务器：
   ```bash
   # 使用Python
   python -m http.server 8000
   
   # 使用Node.js
   npx serve .
   ```

### 部署到服务器
1. 将所有文件上传到服务器根目录
2. 确保 `index.html` 在根目录
3. 配置域名指向服务器IP
4. 访问域名即可查看网站
